## 📋 Описание проекта

**SCP:SL Keycard Generator** - это легковесный веб-интерфейс, разработанный для администраторов серверов игры **SCP: Secret Laboratory (версии 14.1 и выше)**. Инструмент позволяет быстро и безошибочно генерировать внутриигровые консольные команды `ckeycard` для создания кастомных ключ-карт.

Интерфейс выполнен в строгом футуристичном стиле под терминалы Зоны 02 (в духе киберпанка/SCP) с адаптивной версткой и интерактивными элементами.

## ✨ Особенности

* 📦 **Полная автономность:** Проект написан на чистом стеке (HTML5, CSS3, Vanilla JS) - никаких тяжелых фреймворков или внешних зависимостей.
* 💳 **Поддержка всех типов карт:** Корректная генерация синтаксиса для всех четырех игровых типов: `Обычная`, `Админская`, `Оперативника` и `Металлическая`.
* 🎨 **Автоматическая стилизация:** При выборе типа карты система автоматически выставляет базовые HEX-коды цветов для кружков, обводки и текста, которые при желании можно кастомизировать вручную.
* 🔢 **Умные поля:** 
    * Автоматическая генерация 10-значного серийного номера для карт повышенного уровня доступа.
    * Динамическое скрытие/отображение строк параметров (строка серийного номера и количества кружков появляется только тогда, когда это необходимо).
    * Валидация ID игрока (строго цифры, до 5 знаков).
* 📋 **Быстрое копирование:** Кнопка копирования готовой команды в один клик со встроенной системой всплывающих уведомлений (Toasts).
* 🔒 **Конфиденциальность:** Проект полностью очищен от персистентности данных (данные не сохраняются в local storage и сессиях, обеспечивая чистоту работы при каждой перезагрузке).

## 📖 Использование и синтаксис

Интерфейс собирает параметры в официальную команду сервера SCP:SL. 

### Примеры логики генерации:
* **Обычная карта:** `ckeycard [ID] KeycardCustomSite02 [Название] [SCP] [Оружие] [Админ] [Цвет_Кружков] [Цвет_Обводки] [Должность] [Цвет_Текста] [Имя] 1`
* **Карта Оперативника:** Формирует специальный хвост команды, привязывая серийный номер и переводя выбранное количество кружков в инвертированные индексы игры (например: 3 кружка на UI превращаются в индекс `1` для синтаксиса игры).

> ⚠️ **Важно:** Все пробелы в текстовых полях (*Название в инвентаре, Должность, Имя на карте*) при генерации автоматически заменяются на нижнее подчеркивание (`_`), чтобы консоль игры корректно считывала аргументы команды.
